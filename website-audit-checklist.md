# Website Audit Checklist

Use this checklist to evaluate your website and identify improvements.

## UX clarity

- [ ] Is the core value proposition obvious within 5 seconds?
- [ ] Is the primary audience clear from the headline and visuals?
- [ ] Is the navigation simple and easy to follow?
- [ ] Are pages free of unnecessary clutter and distractions?
- [ ] Is the next action clear on each key page?

## page speed

- [ ] Does the homepage load quickly on desktop and mobile?
- [ ] Are large images compressed and correctly sized?
- [ ] Are unused scripts, fonts, or plugins removed?
- [ ] Is caching enabled for static assets?
- [ ] Are Core Web Vitals monitored and improving?

## conversion elements

- [ ] Is there one clear primary call-to-action (CTA) per page?
- [ ] Are CTA buttons visible above the fold?
- [ ] Are forms short and easy to complete?
- [ ] Is trust proof included (testimonials, logos, case studies)?
- [ ] Is there a clear path to contact, demo, or purchase?

## analytics tracking

- [ ] Is analytics installed correctly on all pages?
- [ ] Are key conversion events tracked (form submit, signup, purchase)?
- [ ] Are UTM parameters used consistently for campaigns?
- [ ] Is a dashboard available for weekly KPI review?
- [ ] Are funnels tracked to identify drop-off points?

## SEO basics

- [ ] Does each page have a unique title tag and meta description?
- [ ] Are headings structured properly (one H1, logical H2/H3)?
- [ ] Are URLs clean, readable, and keyword-relevant?
- [ ] Is internal linking used to connect related content?
- [ ] Is the site indexable with a valid sitemap and robots.txt?

## mobile responsiveness

- [ ] Does the layout adapt cleanly across phone and tablet sizes?
- [ ] Is text readable without zooming?
- [ ] Are tap targets large enough and easy to use?
- [ ] Do forms and menus work smoothly on mobile?
- [ ] Are popups/modals mobile-friendly and non-intrusive?
